== Description ==

This plugin places an image placeholder on you're website, using a shortcode from placehold.it. You can easly build or construct a website using the image placeholder.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the folder `image placeholder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
